
  ## Network Topology

This homelab simulates a small enterprise environment in progress with:

- 1 Domain Controller (DC01)
- 1 File Server (FS01)
- 2 Client machines
- A single subnet (192.168.10.0/24) 

## Key Components

| Component | Role | IP Address |
|----------|------|-----------|
| DC01 | Domain Controller | 192.168.10.2 |
| FS01 | File Server | 192.168.10.3 |
| tfennanew11 | admin user Machine | 192.168.10.10 |
| zrobaiiw11 | User Machine | 192.168.10.11 |
